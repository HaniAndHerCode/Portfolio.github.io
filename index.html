Description

My second project is MITM Detection Using Raspberry Pi.
This project is my final year project in semester 5 of my diploma program. It is about when an attacker intercepts communications between two parties either to secretly eavesdrop or modify traffic traveling between this two. Also, Man-in-the-Middle sit between user or user browser and the website user are visiting to intercept and capture any data user submit to the site, such as login credentials or financial information. As Man-in-the-Middle can develop attacks, capture the critical user information and attack user by using DNS spoofing, IP spoofing and ARP spoofing using Unix platform only.


- To develop an attack for attackers in MITM by DNS spoofing, IP spoofing & ARP spoofing using Unix platform only.
- To capture the critical user information such as usernames and password.
- To make a fake website to steal personal information like username and password.
- To attack by using DNS spoofing, IP spoofing & ARP spoofing.
- Technologies used: Kali Linux, Wireshark and Raspberry Pi.
